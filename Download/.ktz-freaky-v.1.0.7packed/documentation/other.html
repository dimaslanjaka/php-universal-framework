1. Step by step installation

If your failed install your theme please check your plugin and uninstall all plugin and then reinstall or install it in your new wordpress installation, this is step by step installation themes:

  • Install themes in your wordpress
  • Install demo xml in your wordpress
  • Create menus
  • Setting your themes in admin panel.
  • Done.
  • Still confuse? Contact us for solve the problem.

2. Translate

To translate your website, use the files located in languages/en_US.po and open it using POEdit and translate the theme strings to your local language..

After that save it and upload them to languages/, for example you saved it as id_ID.po, a new file will be also created id_ID.mo, You must upload both of them to the folder

Now open your WordPress site root and open wp-config.php and make sure that WP_LANG is like this

define('WPLANG', 'id_ID');

That's it, your website is ready for your local language..
