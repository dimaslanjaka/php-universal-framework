2. Installation theme

There are two ways to install this theme: you can upload theme.zip file by “Upload Theme” button in Wordpress. Or, you can copy all files using FTP client.

2.1) Installing through Wordpress Theme Upload

Depending on Wordpress version you are using, there might be some view differences with this following tutorial. This is based on Wordpress 3+.

  • First, extract kentooz-theme.ZIP.
  • Login to your WordPress admin.
  • In "Appearance > Themes" menu, click tab "Install Themes"
  • Click "Upload", then choose file.
  • Find and choose file named kentooz-theme.zip then click "Install Now"
  • After installation is succesful, you will be confirmed about this new theme.
  • Click "Activate" link.

2.2) Uploading file via FTP

  • Login to your FTP server and find your WordPress theme directory.
  • Usually, the location is in "wp-content/themes"
  • Copy and paste "Kentooz theme" folder into the directory.
  • Login to WordPress admin.
  • In "Appearance" menu, click "Themes"
  • Click "Activate" link on theme "kentooz theme"

2.3) Troubleshooting after installation

If you have troubleshooting after installation, please upgrade wordpress to new version, uninstall your plugin and then reinstall the theme.

NB: Kentooz theme name can be kasep, karma, kreview, kapro or whatever name in kentooz downloader.

2.4) How to update theme if available

  • Before update please backup your wordpress site or backup your themes.
  • Replace all kentooz themes folder with latest theme version
  • Done.
  • If any error when update, restore your backup files.

3. Setting up Menus

First, you need to prepare your menu. This theme supports the custom menu feature.

To create menu go to Appearance > Menus. Create the menu by entering title of the menu in the field Menu Name and clicking Save Menu to save.Then activate the menu by choosing active menu on the left and saving it.

  • Go to "Appearance" > "Menus".
  • To create a custom navigation menu, type a name for it (in this tutorial, type Primary menu as a name and click Create Menu.
  • You need to add pages to your menu by checking the proper boxes for the pages you want and then click Add to Menu (You can also adding Category pages, Tag pages and create Custom link.)
  • Be sure to click Save Menu each time you make changes to your custom menu.
  • Create another custom navigation menu, this time type Secondary menu as a name. (follow step 1 - step 4.)
  • Next step is integrate your custom menu into the right place.
  • Click Save.

5. Post setting and thumbnail

Similar with other theme in wordpress, create post very easy, but in kentooz theme support thumbnail and other options to make beautiful posting. Let's see.

  • Create Pages from the "Add New Post" panel.
  • In editor you can see shortcode, we support shortcode for video (youtube and vimeo), googlemap and soundcloud.
  • Now kentooz support BFI_thumb. This script can better crop in the fly, so you can save your resource and bandwith. Our script can detect first image on post, or set featured image in your post, whatever you liked.
  • You can setting your post via metaboxes for example exclusive sidebar, exclusive background, seo, etc.
