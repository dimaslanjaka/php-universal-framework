1. Homepage layout

Below is homepage layout:

Description image above.

  1. Top Menu = You can set in appareances->menus. See tutorial how to set menus.
  2. Logo = You can set in theme options -> general
  3. Top banner = You can set in theme options -> banner
  4. Blog Content = Automatic display in frontpage
  5. Sidebar = You can set via appareances->widget or import default widget. Look here to import default widget
  6. Footer banner = You can set in theme options -> banner
  7. Footer = You can set via appareances->widget or import default widget. Look here to import default widget
  8. Footer Menu = You can set in appareances->menus. See tutorial how to set menus.
  9. Copyright = You can set in theme options -> general
  10. Social icon = You can set in theme options -> social network
