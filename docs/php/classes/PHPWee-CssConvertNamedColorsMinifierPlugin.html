CssConvertNamedColorsMinifierPlugin extends aCssMinifierPlugin

CssMin.php : 2696

This {@link aCssMinifierPlugin} will convert named color values to hexadecimal notation.

Example: color: black; border: 1px solid indigo;

Will get converted to: color:#000; border:1px solid #4b0082;
