ScramblePrivateMethod

Renames private methods

WARNING

This method is not foolproof. This visitor scans for all private method declarations and renames them. It then finds all method calls in the class, and renames them if they match the name of a renamed method. If your class calls a method of another class that happens to match one of the renamed private methods, this visitor will rename it.
