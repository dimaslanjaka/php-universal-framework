\PHPWee

\PHPWeeCssRemoveEmptyRulesetsMinifierFilter

This {@link aCssMinifierFilter minifier filter} will remove any empty rulesets (including @keyframes at-rule block rulesets).

Minifier filters allows a pre-processing of the parsed token to add, edit or delete tokens. Every minifier filter has to extend this class.
